# Language Translation Tool

A simple and interactive web application that translates text between multiple languages using Google Translate.

This project was completed as Task 1 of my AI Internship at CodeAlpha.

---

## Features

- Translate text between English, Hindi, Spanish, French, and German
- Clean and user-friendly interface
- Copy translated text with one click
- Text-to-speech support for translated text
- Fast and accurate translations

---

## Technologies Used

- Python
- Streamlit
- deep-translator (Google Translate API)

---

## How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   streamlit run app.py

3. Open browser at: http://localhost:8501

---

## How to Use

1. Select "Translate from" language
2. Select "Translate to" language
3. Enter text in the input box
4. Click "Translate" button
5. View translated text instantly
